## 发现问题

- 中：学生端“我的申请”页面在教师审批前存在状态不一致。
- 现象：申请刚提交、审批状态仍为“待审批”时，页面中的“实习阶段”已经显示为“已录取”。
- 影响：会让学生误以为已经进入录取阶段，容易造成业务认知错误。
- 复现路径：学生登录后提交项目申请，教师尚未审批前进入“我的申请”页面查看记录。

- 低：多处业务状态仍直接显示英文枚举值。
- 现象：页面中可见 `OPEN`、`PENDING`、`DAILY`、`SUBMITTED`、`APPROVED`、`CONFIRMED` 等英文状态。
- 影响：整体界面以中文为主，这些英文状态会影响一致性和可读性。
