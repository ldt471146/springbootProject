# 学生实习管理系统 - 功能测试指南

## 系统架构
- **后端**: Spring Boot 3.5.11 + MyBatis-Plus + MySQL
- **前端**: Vue 3 + Vite
- **认证**: JWT Token
- **端口**: 8081

## 前置条件

### 1. 环境准备
```bash
# 确保已安装
- Java 17+
- Maven 3.8+
- MySQL 8.0+
- Node.js v20.18.0
- npm 10.8.2
```

### 2. 数据库配置
```bash
# 创建数据库
CREATE DATABASE internship_system CHARACTER SET utf8mb4;

# 修改 application.yml 中的数据库连接信息
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/internship_system
    username: root
    password: your_password
```

### 3. 启动应用
```bash
# 方式1: Maven 启动
mvn clean install
mvn spring-boot:run

# 方式2: IDE 启动
直接运行 SpringBootProjectApplication.java

# 应用启动后访问
- 后端 API: http://localhost:8081/api
- Swagger 文档: http://localhost:8081/doc.html
- 前端应用: http://localhost:8081
```

## 常见问题排查

### 1. 数据库连接失败
```
检查 MySQL 是否运行
检查 application.yml 中的数据库配置
检查数据库用户权限
```

### 2. 前端无法加载
```
检查 npm install 是否完成
检查 npm run build 是否成功
检查 8081 端口是否被占用
```

### 3. JWT Token 过期
```
Token 有效期: 24 小时
重新登录获取新 Token
检查系统时间是否正确
```

### 4. 文件上传失败
```
检查 uploads 目录是否存在
检查文件大小是否超过限制（20MB）
检查文件类型是否被允许
```
